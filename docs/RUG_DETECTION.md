# Rug Detection

Rug detection is implemented as a transparent evidence model, never a black-box override.

Current evidence sources include:

- early dev selling
- 70% launch collapse checks
- no-buy-gap detection
- holder-growth stall detection
- sell-dominance and concentration pressure
- profit-overhang and free-roll dump pressure
- funding-graph suspicion
- fake-momentum evidence
- data-quality penalties

The engine recommends keep, soft discard, hard discard, rug archive, or data-gap stop without claiming certainty.
